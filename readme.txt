# **Orowan Project - Tiger team 6**

## Project Setup

1. Assurez vous d'avoir H2 installé sur votre ordinateur.
2. Déplacer le fichier "data" à l'emplacement C://Users/<Votre nom d'utilisateur>
3. Lancez le fichier run.bat
